# AI随身场景化笔记助手

一款轻量级的微信小程序笔记应用，专为学生和职场人士设计，提供多样化的笔记录入方式和智能管理功能。

## 项目简介

本项目是为全国大学生创新比赛开发的AI笔记助手，提供以下核心能力：
- **多样化录入**：拍照笔记、手写涂鸦、长文总结、模版笔记
- **智能管理**：笔记编辑、分类、搜索、导出、分享
- **便捷设置**：标签管理、数据管理、账号管理

## 核心功能

### 1. 多样化录入方式
- **拍照记笔记**：拍照并添加文字备注，快速记录现场信息
- **手写涂鸦**：Canvas绘画板，支持多种颜色和笔触，自由创作保存
- **长文总结**：AI智能总结，提供三种模式
  - 简要总结：快速提炼核心要点
  - 详细总结：保留重要细节的完整总结
  - 要点提取：结构化列出关键信息
- **模版笔记**：三种预设模板，快速创建结构化笔记
  - 会议记录模板
  - 学习笔记模板
  - 待办清单模板

### 2. 笔记管理
- **笔记详情页**：
  - 编辑笔记内容
  - 一键复制全文
  - 导出为文本文件
  - 分享到微信好友/群聊
  - 删除笔记
- **搜索功能**：全文搜索已有笔记，支持标题和内容匹配
- **标签分类**：自定义标签，便于笔记分类管理

### 3. 设置与管理
- **标签管理**：添加、编辑、删除自定义标签
- **数据管理**：导出所有笔记、清除缓存数据
- **关于应用**：查看版本信息和使用说明
- **账号管理**：用户信息和权限设置

## 技术栈

- **框架**：微信小程序原生开发
- **AI能力**：文本总结、智能分类
- **绘图能力**：Canvas 2D API
- **数据存储**：微信本地存储（wx.setStorageSync）
- **图片处理**：微信相机API、图片压缩

## 快速开始

### 1. 环境准备

- 安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号，获取AppID

### 2. 配置项目

1. 克隆或下载本项目到本地
2. 使用微信开发者工具打开项目目录
3. 修改 `project.config.json` 中的 `appid` 为你的小程序AppID

### 3. 运行项目

1. 在微信开发者工具中点击"编译"
2. 在模拟器或真机预览中测试功能

## 部署说明

### 1. 代码审核前检查

- [ ] 测试所有核心功能正常运行
- [ ] 确认用户隐私协议和权限说明
- [ ] 测试真机兼容性

### 2. 上传代码

1. 在微信开发者工具中点击"上传"
2. 填写版本号和项目备注
3. 上传成功后在微信公众平台查看

### 3. 提交审核

1. 登录[微信公众平台](https://mp.weixin.qq.com/)
2. 进入"版本管理" -> "开发版本"
3. 选择刚上传的版本，点击"提交审核"
4. 填写审核信息：
   - 功能页面：首页、拍照记笔记、手写涂鸦、长文总结、模版笔记、笔记详情、搜索、设置
   - 测试账号：提供测试用的微信号
   - 补充说明：说明各功能的使用场景

### 4. 发布上线

审核通过后，在"版本管理"中点击"发布"即可上线。

## 权限说明

本小程序需要以下权限：

- **相机权限**（scope.camera）：用于拍照记笔记
- **相册权限**（scope.writePhotosAlbum）：用于保存手写涂鸦图片

## 注意事项

### 1. 数据存储

- 当前使用微信本地存储，单个小程序限制10MB
- 建议定期导出笔记备份数据
- 可在设置页面清除缓存释放空间

### 2. 性能优化

- 图片上传前进行压缩
- 长列表使用虚拟列表优化
- 搜索使用防抖处理

### 3. 兼容性

- 建议在真机上测试所有功能
- Canvas绘图功能需要微信版本7.0.0以上
- 部分功能在模拟器上可能表现不一致

## 常见问题

### Q1: 拍照功能无法使用？
A: 确认已授权相机权限，在真机上测试（模拟器可能不支持）。

### Q2: 手写涂鸦保存失败？
A: 确认已授权相册权限，检查存储空间是否充足。

### Q3: 数据存储满了？
A: 在设置页面清除缓存或导出笔记后清空数据。

### Q4: 如何备份笔记？
A: 在设置页面使用"导出所有笔记"功能，将笔记导出为文本文件。

### Q5: 长文总结功能如何使用？
A: 输入或粘贴长文本后，选择总结模式（简要/详细/要点），系统会自动生成总结。

# AI随身场景化笔记助手

一款轻量级的微信小程序笔记应用，专为学生和职场人士设计，解决笔记杂乱、整理耗时、检索困难的问题。

## 项目简介

本项目是为创新竞赛开发的AI笔记助手，聚焦三大核心功能：
- **即时录入**：支持语音、拍照、手写三种输入方式
- **AI智能整理**：自动分类、提取关键词、生成标题、识别待办事项
- **即时检索**：支持文字和语音搜索，快速定位笔记

## 核心功能

### 1. 即时录入
- **语音录入**：支持暂停/继续录音，自动语音转文字
- **拍照录入**：支持拍照或相册选择，最多3张图片，批量OCR识别
- **手写录入**：Canvas手写板，支持多种颜色和笔触，手写识别

### 2. AI智能整理
- 自动分类（学习、工作、生活、灵感等可自定义标签）
- 关键词提取
- 智能标题生成
- 会议笔记处理（提取参与者、待办事项）

### 3. 即时检索
- 全文搜索，支持标题、内容、关键词匹配
- 语音搜索支持
- 搜索历史记录
- 热门搜索推荐
- 关键词高亮显示

### 4. 其他功能
- 笔记锁定（密码保护）
- 笔记导出（文本格式）
- 分享到微信
- 标签管理
- 数据导出

## 技术栈

- **框架**：微信小程序原生开发
- **OCR识别**：百度OCR API（通用文字识别、手写文字识别）
- **语音识别**：微信原生RecorderManager + 语音转文字API
- **数据存储**：微信本地存储（wx.setStorageSync）
- **AI文本分析**：关键词提取、文本分类、待办事项识别

## 快速开始

### 1. 环境准备

- 安装[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 注册微信小程序账号，获取AppID

### 2. 配置项目

1. 克隆或下载本项目到本地
2. 使用微信开发者工具打开项目目录
3. 修改 `project.config.json` 中的 `appid` 为你的小程序AppID

### 3. 配置API密钥

#### 百度OCR配置

1. 注册[百度智能云](https://cloud.baidu.com/)账号
2. 创建文字识别应用，获取API Key和Secret Key
3. 在小程序设置页面配置密钥，或直接修改 `utils/api.js`：

```javascript
// utils/api.js
const apiConfig = wx.getStorageSync('apiConfig') || {
  baiduApiKey: '你的API_KEY',
  baiduSecretKey: '你的SECRET_KEY'
}
```

#### 服务器域名配置

在微信小程序后台配置以下request合法域名：
- `https://aip.baidubce.com`（百度OCR）

### 4. 运行项目

1. 在微信开发者工具中点击"编译"
2. 在模拟器或真机预览中测试功能

## 权限说明

本小程序需要以下权限：

- **录音权限**（scope.record）：用于语音录入和语音搜索
- **相机权限**（scope.camera）：用于拍照录入
- **相册权限**（scope.writePhotosAlbum）：用于保存图片
- **位置权限**（scope.userLocation）：可选，用于地理位置标记

## 注意事项

### 1. API配额限制

- 百度OCR免费版：每日500次调用
- 建议在生产环境升级为付费版本或实现调用次数限制

### 2. 数据存储

- 当前使用微信本地存储，单个小程序限制10MB
- 建议后续接入云开发或自建后端服务

### 3. 语音识别

- 当前使用模拟数据，需要接入真实的语音识别服务
- 可选方案：微信同声传译插件、讯飞语音、腾讯云ASR

### 4. 性能优化

- 图片上传前进行压缩
- 长列表使用虚拟列表优化
- 搜索使用防抖处理

## 常见问题

### Q1: OCR识别失败？
A: 检查API密钥配置是否正确，确认服务器域名已添加到白名单。

### Q2: 语音录音无法使用？
A: 确认已授权录音权限，在真机上测试（模拟器可能不支持）。

### Q3: 数据存储满了？
A: 在设置页面清除缓存或导出笔记后清空数据。

### Q4: 如何备份笔记？
A: 在设置页面使用"导出所有笔记"功能，将笔记导出为文本文件。
